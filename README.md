# Puzzles
Puzzles项目的创建目的是为了便于初创团队快速建立适合业务所需的管理后台而创建的项目。这个项目的核心思想就是希望用户可以掌握基础的编程知识之后，就能够快速构建自己所需的业务CMS平台。为了达到这个目的，Puzzles和其子项目可以帮助用户简单地引入所需的开发工具或框架，并且提供足够的规范支持，便于大家可以快速开发所需的业务模块.
